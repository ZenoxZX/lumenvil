# Lumenvil

> *Where builds come to light*

Self-hosted Unity build automation system for PC/Steam games.

## What it does

- Automates Unity builds
- Real-time build monitoring via web dashboard
- Git integration (clone, pull, branch selection)
- Steam deployment via SteamCMD

## Tech Stack

- **Backend**: .NET 8 Web API + SignalR
- **Build Agent**: .NET 8 (Windows Service)
- **Dashboard**: Next.js 14 + TypeScript
- **Database**: SQLite
